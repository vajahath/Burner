Bulk SMS Gateway API

bms.com allows users to send bulk SMS text messages, and manage interactive SMS based services directly via our online web interface. In addition, developers can link directly to the our gateway via a HTTP Post. This can be done either via standard non-encrypted port 80 requests or via an encrypted connection using SSL. This gives developers the opportunity to add interactive sms capabilities to their existing systems using bms purely as an behind the scenes SMS gateway service.

To setup an bms account Please contact us.
